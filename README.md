# Gihansa Senukie Portfolio

Personal portfolio for Gihansa Senukie, a software engineering undergraduate building full-stack and mobile projects and looking for software engineering internship opportunities.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Resend

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Environment Variables

Create a `.env.local` file in the project root with:

```bash
RESEND_API_KEY=your_api_key_here
CONTACT_EMAIL=your_email@example.com
```

`RESEND_API_KEY` is used by the contact form backend, and `CONTACT_EMAIL` is the inbox that receives portfolio contact messages.
